# Security Policy

## Supported Versions

Planetary only provide support and security fixes for the latest builds.
We always recommend keeping your Instance up-to-date to ensure you have the best user experience.

| Version | Supported          |
| ------- | ------------------ |
| 2.1.x   | :white_check_mark: |

## Reporting a Vulnerability

Any security vulnerabilities should be raised to our Engineering team.
To allow us to promptly address the concern, please provide us with:

- Version ID
- Browser Console Logs
- Browser Network Logs
- PM2 / NPM Logs
- Description

E: IT@planetaryapp.us
